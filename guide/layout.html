布局组件 ​

ResponsiveContainer、ResponsiveGrid、ResponsiveFlex、ResponsiveStack 提供跨框架一致的布局能力。本章节将逐一介绍其用途、核心属性、以及在 Vue / React 中的示例。

1. ResponsiveContainer ​

属性类型默认值说明
as`stringComponent`div
managerResponsiveManager自动创建自定义 runtime，常用于多容器
initialSnapshotResponsiveSnapshotInput-覆盖容器初始状态

Vue 示例：

vue
<script setup lang="ts">
import { ResponsiveContainer, useResponsive } from "@tlgf/responsive";

const rs = useResponsive();
</script>

<template>
  <ResponsiveContainer class="page">
    <h2>当前断点：&#123;&#123; rs.breakpoint &#125;&#125;</h2>
    <slot />
  </ResponsiveContainer>
</template>

React 示例：

tsx
import { ResponsiveContainer } from "@tlgf/responsive/react";

export function Page({ children }) {
  return <ResponsiveContainer className="page">{children}</ResponsiveContainer>;
}

在大屏场景可创建多个 container，通过不同 designWidth 控制各区块缩放。

2. ResponsiveGrid ​

属性类型默认值说明
columns`numberResponsiveColumns`4
gap`numberResponsiveGap`16
minWidthnumber-单列最小宽度，配合自动列数
as`stringComponent`div

ResponsiveColumns 示例：

ts
const columns = {
  base: 1,
  sm: 2,
  md: 3,
  lg: 4,
};

Vue 模板：

vue
<ResponsiveGrid :columns="{ base: 1, md: 2, lg: 4 }" :gap="{ base: 12, md: 20 }">
  <Card v-for="item in items" :key="item.id">&#123;&#123; item.title &#125;&#125;</Card>
</ResponsiveGrid>

React：

tsx
<ResponsiveGrid columns=&#123;&#123; base: 1, md: 3 &#125;&#125; gap=&#123;&#123; base: 12, lg: 24 &#125;&#125;>
  {items.map((item) => (
    <Card key={item.id}>{item.title}</Card>
  ))}
</ResponsiveGrid>

3. ResponsiveFlex ​

ResponsiveFlex 将常见的「左右固定 + 中间自适应」布局抽象为统一组件，并在窄屏下自动折叠。

属性类型默认值说明
gap / gapX / gapYnumber24主轴及横纵向间距，单位 px
direction"row" | "row-reverse" | "column" | "column-reverse""row"主轴方向
align / justifystring"stretch" / "flex-start"与原生 Flex 语义一致
wrapbooleanfalse是否允许子元素换行
collapseAtnumber0容器宽度小于等于该值时折叠为纵向布局
startWidth / endWidthnumber | string320左/右区域固定宽度，支持 %
centerMinWidth / centerMaxWidthnumber | string0 / undefined中间区域宽度约束
fullHeight（React）booleanfalse让容器最小高度占满父级
actions（插槽 / prop）VNode | ReactNode-右侧操作区，优先级高于 end

插槽 / Prop 对应关系：

  • start：左侧固定列
  • center：中间自适应区域
  • end：右侧固定列
  • actions：右侧操作区域（Vue #actions / React actions）
  • 默认插槽 / children：未声明命名插槽时直接渲染子节点

当触发折叠时，容器会附加 data-rs-collapsed="true"，可在样式中做差异化处理。

vue
<ResponsiveFlex :collapse-at="768" :center-min-width="240">
  <template #start>
    <Sidebar />
  </template>
  <template #center>
    <Article />
  </template>
  <template #actions>
    <Toolbar />
  </template>
</ResponsiveFlex>
tsx
<ResponsiveFlex
  collapseAt={768}
  start={<Sidebar />}
  center={<Article />}
  actions={<Toolbar />}
  fullHeight
/>

4. ResponsiveStack ​

适合仪表盘分区、表单区块等纵向堆叠的场景，可在宽屏下自动横向排列，并支持子项折叠。

属性类型默认值说明
gapnumber20子项之间的间距
itemGapnumber12ResponsiveStackItem 内部的行间距
dividerbooleanfalse是否渲染分隔线样式
horizontalAtnumber0宽度大于该值时自动横向排列
collapseAtnumber0宽度小于等于该值时启用卡片样式
itemRadiusnumber16折叠态卡片圆角
itemShadowstring"0 12px 30px rgba(34, 41, 82, 0.08)"折叠态卡片阴影

ResponsiveStackItem 还提供以下能力：

属性类型默认值说明
titlestring-标题文本
collapsiblebooleanfalse是否允许折叠
defaultOpenbooleantrue初始展开状态
插槽header / icon-自定义标题区域与折叠图标

组件会在根节点写入数据属性，便于在样式中钩入：

  • data-rs-horizontal="true"：当前为横向布局
  • data-rs-collapsed="true"：处于折叠卡片样式
vue
<ResponsiveStack :collapse-at="640" :horizontal-at="1200" divider>
  <ResponsiveStackItem title="订单概览" collapsible>
    <OrderSummary />
  </ResponsiveStackItem>
  <ResponsiveStackItem collapsible :default-open="false">
    <template #header="{ open }">
      <h3>销售漏斗</h3>
      <span>{{ open ? "收起" : "展开" }}</span>
    </template>
    <template #icon="{ open }">
      <Chevron :open="open" />
    </template>
    <FunnelChart />
  </ResponsiveStackItem>
</ResponsiveStack>
tsx
<ResponsiveStack collapseAt={640} horizontalAt={1200}>
  <ResponsiveStackItem title="Orders" collapsible>
    <OrderSummary />
  </ResponsiveStackItem>
  <ResponsiveStackItem collapsible defaultOpen={false}>
    <DashboardSection />
  </ResponsiveStackItem>
</ResponsiveStack>

响应式属性格式 ​

布局组件使用像素级别的简单配置。若需根据断点动态切换值，可借助 useResponsive() 返回的 breakpoint 与 px()/rem() 辅助函数，在模板或样式中自行判断：

ts
const rs = useResponsive();
const gutter = computed(() => (rs.breakpoint.value === "xl" ? 32 : 16));

也可以搭配 styled，在模板字符串内访问 rs 上下文实现更复杂的断点逻辑。

样式结合 ​

组合 styled 与布局组件：

ts
const Panel = styled(ResponsiveFlex)`
  background: var(--rs-surface);
  border-radius: 20px;
  padding: calc(24px * var(--rs-scale));
`;

借助 styled，可以在保持响应式断点能力的同时定制外观。

最佳实践 ​

  • PC 列表页：ResponsiveGrid + ResponsiveContainer，根据断点动态调整列数
  • 移动端内容流：ResponsiveFlex 设置 direction 切换行/列，实现单列到双列的平滑过渡
  • 数据大屏：ResponsiveStack 拆分区域，配合 position: absolute 的 Panel 完成分区布局
