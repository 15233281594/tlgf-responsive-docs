布局组件 ​

ResponsiveContainer、ResponsiveGrid、ResponsiveFlex、ResponsiveStack 提供跨框架一致的布局能力。本章节将逐一介绍其用途、核心属性、以及在 Vue / React 中的示例。

1. ResponsiveContainer ​

属性类型默认值说明
as`stringComponent`div
managerResponsiveManager自动创建自定义 runtime，常用于多容器
initialSnapshotResponsiveSnapshotInput-覆盖容器初始状态

Vue 示例：

vue
<script setup lang="ts">
import { ResponsiveContainer, useResponsive } from "@tlgf/responsive";

const rs = useResponsive();
</script>

<template>
  <ResponsiveContainer class="page">
    <h2>当前断点：&#123;&#123; rs.breakpoint &#125;&#125;</h2>
    <slot />
  </ResponsiveContainer>
</template>

React 示例：

tsx
import { ResponsiveContainer } from "@tlgf/responsive/react";

export function Page({ children }) {
  return <ResponsiveContainer className="page">{children}</ResponsiveContainer>;
}

在大屏场景可创建多个 container，通过不同 designWidth 控制各区块缩放。

2. ResponsiveGrid ​

属性类型默认值说明
columns`numberResponsiveColumns`4
gap`numberResponsiveGap`16
minWidthnumber-单列最小宽度，配合自动列数
as`stringComponent`div

ResponsiveColumns 示例：

ts
const columns = {
  base: 1,
  sm: 2,
  md: 3,
  lg: 4,
};

Vue 模板：

vue
<ResponsiveGrid :columns="{ base: 1, md: 2, lg: 4 }" :gap="{ base: 12, md: 20 }">
  <Card v-for="item in items" :key="item.id">&#123;&#123; item.title &#125;&#125;</Card>
</ResponsiveGrid>

React：

tsx
<ResponsiveGrid columns=&#123;&#123; base: 1, md: 3 &#125;&#125; gap=&#123;&#123; base: 12, lg: 24 &#125;&#125;>
  {items.map((item) => (
    <Card key={item.id}>{item.title}</Card>
  ))}
</ResponsiveGrid>

3. ResponsiveFlex ​

属性类型默认值说明
direction`ResponsiveValue<"row""column">`row
wrap`ResponsiveValue<boolean"wrap""nowrap">`
gapResponsiveGap12主轴/交叉轴间距
align, justifyResponsiveValue<FlexAlign>-对齐方式

示例：

vue
<ResponsiveFlex
  :direction="{ base: 'column', md: 'row' }"
  :gap="{ base: 12, md: 24 }"
  :align="{ base: 'stretch', md: 'center' }"
>
  <aside class="sidebar" />
  <main class="content" />
</ResponsiveFlex>

React 示例：

tsx
<ResponsiveFlex
  direction=&#123;&#123; base: "column", lg: "row" &#125;&#125;
  gap=&#123;&#123; base: 16, lg: 32 &#125;&#125;
  align=&#123;&#123; base: "stretch", lg: "center" &#125;&#125;
>
  <FilterPanel />
  <ResultList />
</ResponsiveFlex>

4. ResponsiveStack ​

用于垂直或水平堆叠元素，支持分隔符与 spacing 控制。

属性类型默认值说明
direction`ResponsiveValue<"horizontal""vertical">`vertical
spacingResponsiveGap12节点间距
divider`VNodeReactNode`-

Vue 示例：

vue
<ResponsiveStack :spacing="{ base: 12, md: 20 }" divider="<hr />">
  <SectionA />
  <SectionB />
  <SectionC />
</ResponsiveStack>

React 示例：

tsx
<ResponsiveStack spacing=&#123;&#123; base: 16, lg: 24 &#125;&#125; divider={<Divider />}>
  <SectionA />
  <SectionB />
</ResponsiveStack>

响应式属性格式 ​

所有 ResponsiveValue<T> 支持以下写法：

  • 单值：gap={16}
  • 对象映射断点：gap=&#123;&#123; base: 12, md: 20, xl: 32 &#125;&#125;
  • 函数：gap={(ctx) => (ctx.breakpoint === "xl" ? 40 : 16)}

断点来源为 runtime 的 breakpoints 定义，可通过插件选项自定义。

样式结合 ​

组合 styled 与布局组件：

ts
const Panel = styled(ResponsiveFlex)`
  background: var(--rs-surface);
  border-radius: 20px;
  padding: calc(24px * var(--rs-scale));
`;

借助 styled，可以在保持响应式断点能力的同时定制外观。

最佳实践 ​

  • PC 列表页：ResponsiveGrid + ResponsiveContainer，根据断点动态调整列数
  • 移动端内容流：ResponsiveFlex 设置 direction 切换行/列，实现单列到双列的平滑过渡
  • 数据大屏：ResponsiveStack 拆分区域，配合 position: absolute 的 Panel 完成分区布局
