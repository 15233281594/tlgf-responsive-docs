Masonry 指南 ​

ResponsiveMasonry 是跨框架的瀑布流组件，内置虚拟滚动、懒加载、列权重缓存等特性。本节将介绍使用方式、关键参数与调优策略。

基本使用 ​

Vue ​

vue
<script setup lang="ts">
import { ResponsiveMasonry, styled } from "@tlgf/responsive";

const Card = styled.div`
  border-radius: 16px;
  background: var(--rs-surface);
  box-shadow: 0 6px 18px rgba(15, 23, 42, 0.12);
`;

const items = Array.from({ length: 60 }, (_, i) => ({
  id: `photo-${i}`,
  height: 200 + (i % 5) * 40,
}));
</script>

<template>
  <ResponsiveMasonry
    :items="items"
    :columns="{ base: 2, md: 3, xl: 5 }"
    :gutter="{ base: 12, md: 20 }"
    :virtual="true"
    :overscanForward="200"
    trackBy="id"
    v-slot="{ item }"
  >
    <Card :style="{ height: `${item.height}px` }" />
  </ResponsiveMasonry>
</template>

React ​

tsx
import { ResponsiveMasonry, styled } from "@tlgf/responsive/react";

const Card = styled.div`
  border-radius: 16px;
  background: var(--rs-surface);
  box-shadow: 0 6px 18px rgba(15, 23, 42, 0.12);
`;

const items = Array.from({ length: 60 }, (_, i) => ({
  id: `photo-${i}`,
  height: 200 + (i % 5) * 40,
}));

export function Gallery() {
  return (
    <ResponsiveMasonry
      items={items}
      columns=&#123;&#123; base: 2, md: 4 &#125;&#125;
      gutter=&#123;&#123; base: 12, md: 24 &#125;&#125;
      virtual
      overscanForward={240}
      trackBy="id"
      renderItem={({ item }) => <Card style=&#123;&#123; height: `${item.height}px` &#125;&#125; />}
    />
  );
}

关键参数 ​

参数类型默认值说明
itemsT[]必填数据源，建议包含唯一 id
trackBy`string(item) => string`"id"
columns`numberResponsiveColumns(ctx) => number`
gutter`numberResponsiveGap`16
columnWeightsnumber[]-每列权重，默认平均
virtualbooleanfalse启用虚拟滚动
lazyloadbooleanfalse使用 IntersectionObserver 延迟渲染
overscan, overscanForward, overscanBackwardnumber160额外渲染的高度 (px)
destroyInvisiblebooleanfalse隐藏不可见卡片减小 DOM
renderAheadnumber1额外预渲染卡片数量
scrollContainer`HTMLElementWindow`自动
preserveScrollbooleanfalse重新挂载时保持滚动位置
onVisibleChange(keys: string[]) => void-可见元素变化回调

方法与引用 ​

通过 ref / useRef 获取实例方法：

ts
const masonryRef = ref<ResponsiveMasonryExpose>();

// Vue
masonryRef.value?.scrollToItem("photo-20", {
  behavior: "smooth",
  offset: 100,
});

// React
const masonryRef = useRef<ResponsiveMasonryExpose>(null);
masonryRef.current?.getLayoutSnapshot();

ResponsiveMasonryExpose 包括：

  • scrollToItem(key, options?)
  • getLayoutSnapshot()
  • recomputeLayout()
  • refreshVirtualState()

虚拟滚动与懒加载 ​

  • 开启 virtual 后仅渲染视窗内卡片，适合 500+ 数据
  • lazyload + destroyInvisible 降低图片首次加载压力
  • 结合 overscanForward 调整预渲染距离，滚动速度快时适当增大

列权重与高度缓存 ​

  • columnWeights 允许非均匀列宽：[1.2, 1, 0.8]
  • 内部维护 columnHeights 缓存，快速定位最短列
  • 调试时可通过 getLayoutSnapshot() 查看列分布

使用场景建议 ​

场景建议配置
PC 图文瀑布流columns=\{\{ base: 2, lg: 4 \}\}, virtual, overscanForward=240
移动端内容流columns=\{\{ base: 2, md: 3 \}\}, minHeight 控制卡片区间
大屏媒体墙columnWeights=[1.2, 1, 1], preserveScroll, renderAhead=2

调试技巧 ​

  1. 开启 performance 面板观察 Long Task
  2. 使用 data-testid="masonry-ready" 等标记结合 e2e 测试
  3. 利用 onVisibleChange 统计曝光日志

常见问题 ​

  • 滚动抖动：适当增大 overscanBackward，或减小卡片高度差异
  • 首屏闪动：SSR 场景下结合 suppressInitialDomWrite 与 flushDom()
  • 图片加载过慢：配合懒加载 + 低分辨率占位图，提高用户体验
