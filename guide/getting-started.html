快速开始 ​

本文提供完整的安装与初始化流程，覆盖 Vue 3、Vue 2、React，以及面向 PC / 移动端 / 大屏的基础配置建议。

安装依赖 ​

bash
npm install @tlgf/responsive

如需按需加载 polyfill，可额外安装：

bash
npm install @juggle/resize-observer intersection-observer smoothscroll-polyfill

Vue 3 ​

  1. 注册插件：
ts
// main.ts
import { createApp } from "vue";
import App from "./App.vue";
import { createResponsivePlugin } from "@tlgf/responsive";
import "@tlgf/responsive/styles.css";

const app = createApp(App);
app.use(createResponsivePlugin({ designWidth: 1440 }));
app.mount("#app");
  1. 消费状态：
vue
<script setup lang="ts">
import { useResponsive, styled } from "@tlgf/responsive";

const rs = useResponsive();
const Wrapper = styled.div`
  padding: calc(24px * var(--rs-scale));
`;
</script>

<template>
  <Wrapper>
    当前断点：&#123;&#123; rs.breakpoint &#125;&#125;，宽度：&#123;&#123;
    rs.viewportWidth &#125;&#125;
  </Wrapper>
</template>

Vue 2 ​

  1. 安装插件（Composition API 将自动挂载）：
ts
import Vue from "vue";
import { createResponsivePlugin } from "@tlgf/responsive";
import "@tlgf/responsive/styles.css";

Vue.use(createResponsivePlugin({ designWidth: 1280 }));
  1. 在组件中使用：
vue
<script>
import { defineComponent } from "vue";
import { useResponsive, styled } from "@tlgf/responsive";

const Panel = styled.div`
  padding: calc(16px * var(--rs-scale));
`;

export default defineComponent({
  setup() {
    const rs = useResponsive();
    return { rs };
  },
  components: { Panel },
});
</script>

<template>
  <Panel>
    <p>viewportWidth: &#123;&#123; rs.viewportWidth &#125;&#125;</p>
  </Panel>
</template>

提示：Vue 2 中 styled 组件需 components 注册，可结合 script setup <script setup> 配合 defineExpose。 插件会尝试自动 Vue.use(@vue/composition-api)；若你的项目以 CDN 或其他方式加载 Composition API，可放心复用现有实例。

React ​

  1. 包裹根组件：
tsx
import React from "react";
import ReactDOM from "react-dom/client";
import App from "./App";
import { ResponsiveProvider } from "@tlgf/responsive/react";
import "@tlgf/responsive/styles.css";

ReactDOM.createRoot(document.getElementById("root")!).render(
  <ResponsiveProvider options={{ designWidth: 1440 }}>
    <App />
  </ResponsiveProvider>
);

ResponsiveProvider 会浅比较 options 内容，保持同一个对象引用即可避免不必要的 manager 重建；若需要动态调整参数，可只更新实际变动的字段。 CSS 引入说明
npm 包版本请统一引用 @tlgf/responsive/styles.css。该路径由 exports 字段暴露，Vite / Webpack / Rspack 等现代构建工具都能直接解析。仓库内的示例项目为方便热更新使用了 ../../responsive/src/styles.css，发布时请切换回包内入口以便按需打包与版本管理。
